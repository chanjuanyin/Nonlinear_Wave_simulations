# Simulation 01

For Simulation 01, the wave equation is:

\begin{aligned}
\begin{cases}
    \partial_{tt}u(z,t)-(i)^2\Delta u(z,t) = -u(z,t) + (u(z,t))^3 \qquad (z,t) \in \mathbb{C} \times (0,\infty) \\
    u(z,0) = \phi(z) = \tanh(\frac{i}{\sqrt{6}}z)\\
    \partial_t u(z,0) = \psi(z) = -\sqrt{\frac{2}{3}}sech^2(\frac{i}{\sqrt{6}}z)
\end{cases}
\end{aligned}
