# Simulation 01

For Simulation 01, the wave equation is:

$\frac{\partial^2 u}{\partial t^2}(z,t)-\frac{\partial^2 u}{\partial z^2}(z,t) = (u(z,t))^2$

$u(z,0) = \phi(z) = 6z^{-2}$

$\frac{\partial u}{\partial t}(z,0) = \psi(z) = -12\sqrt{2}z^{-3}$

The analytical solution is:

$u(z,t) = 6(z+\sqrt{2}t)^{-2}$

We set $c=1$, $z=3+0i$, $\lambda = 0.25$, and sample $t \in [0,2.0]$.
