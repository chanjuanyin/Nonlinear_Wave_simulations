# Simulation 02

For Simulation 02, the wave equation is:

$\frac{\partial^2 u}{\partial t^2}(z,t)-\frac{\partial^2 u}{\partial z^2}(z,t) = (u(z,t))^3$

$u(z,0) = \phi(z) = \sqrt{2}z^{-1}$

$\frac{\partial u}{\partial t}(z,0) = \psi(z) = -2z^{-2}$

The analytical solution is:

$u(z,t) = \sqrt{2}(z+\sqrt{2}t)^{-1}$

We set $c=1$, $z=6+0i$, $\lambda = 0.25$, and sample $t \in [0,5.0]$.
